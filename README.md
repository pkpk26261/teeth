# 🦷 牙齒色彩分析系統

一個專為牙科診所設計的網頁應用程式，用於管理病患資料與分析牙齒色彩。

## 📋 功能特色

### 病患管理
- **新增/編輯病患資料**：記錄病患姓名、病歷號等基本資訊
- **病患列表**：快速瀏覽與搜尋所有病患
- **搜尋功能**：透過姓名或病歷號快速查找病患

### 牙齒色彩分析
- **色彩分析工具**：分析牙齒色彩數據
- **視覺化介面**：直覺的操作介面設計

### 資料管理
- **本機儲存**：使用 IndexedDB 將資料安全儲存在本機
- **匯入/匯出功能**：支援資料備份與還原
- **草稿自動儲存**：防止編輯中的資料遺失

## 🚀 快速開始

### 使用方式

1. 直接在瀏覽器中開啟 `index.html` 檔案
2. 或使用本地伺服器運行：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需安裝 http-server)
npx http-server
```

3. 在瀏覽器中訪問 `http://localhost:8000`

## 🛠️ 技術棧

- **HTML5** - 頁面結構
- **CSS3** - 樣式設計（含 CSS 變數）
- **JavaScript (ES6+)** - 應用程式邏輯
- **IndexedDB** - 本機資料儲存
- **JSZip** - ZIP 檔案匯出功能

## 💡 使用說明

### 新增病患
1. 點擊「➕ 新增病患」按鈕
2. 填寫病患基本資料
3. 儲存病患資訊

### 搜尋病患
- 在搜尋欄位輸入病患姓名或病歷號即可快速篩選

### 資料備份
- 點擊「📤 匯出」可將所有資料匯出備份
- 點擊「📥 匯入」可還原先前匯出的資料
